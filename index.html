Software Development and Engineering

One of the biggest areas of growth for me has been applying Software Design and Engineering principles, along with the Software Development Life Cycle (SDLC), to create scalable and maintainable applications. Working with different programming languages, frameworks, and development methodologies has allowed me to build software that is not only functional but also efficient and adaptable.

Capstone Project: Inventory Management App

A great example of this is my CS 499 Capstone project—an inventory management app that brings together everything I’ve learned. Developing this app challenged me to implement optimized algorithms for data processing, ensuring smooth performance even under heavy loads. It also reinforced my skills in database design and management, as I worked with both SQL and NoSQL technologies to create a secure, reliable system. Seeing this project come together has been incredibly rewarding, and I’m proud of how it showcases my ability to design and develop a well-structured, high-performing application.
