Inventory Tracker - Android Studio Mobile Application

This artifact is an app I created Fall 2024 and enhanced Spring of 2025 for the CS499 course at SNHU. The purpose of the app is to allow a user to register for and login with their own credentials to view their invetory lists. The app allows the user to add or remove items with a quanitty and custom title. The app also allows the user to sort the items by date added and by alphabetical order. Users logins are stored in a SQLite database as well as the items and their attributes.
